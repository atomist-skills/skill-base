# `@atomist/skill-base`

Docker base image for JavaScript based Atomist skills

## Usage

Use the following `FROM` line in your Dockerfile:

```
FROM atomist/skill:node14
```

## Use the `atomist/docker-base-image-policy` to pin the `FROM` to latest digests.
